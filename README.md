# IMDB Sentiment Analysis App

This is a simple machine learning app that predicts whether an IMDB movie review is **positive** or **negative** using a trained model.

---

## Files in this repository

- `app.py` – The main Python script to run the app.  
- `imdb_sentiment_model.pkl` – The trained ML model used for predictions.  
- `vectorizer.pkl` – The text vectorizer used to transform reviews before prediction.  

---

## Requirements

You need Python installed. The app uses the following packages:

- streamlit  
- pandas  
- scikit-learn  

Install them manually:

```bash
pip install streamlit pandas scikit-learn
